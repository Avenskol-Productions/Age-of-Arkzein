probe¶

Tags: adventure | fort | inspection | buildings | map | units

Command: probe

Display low-level properties of the selected tile.

Command: bprobe

Display low-level properties of the selected building.

Command: cprobe

Display low-level properties of the selected unit.

Usage¶

probe

Displays properties of the tile selected with k. Some of these properties can be passed into tiletypes.

bprobe

Displays properties of the building selected with q or t. For deeper inspection of the building, see gui/gm-editor.

cprobe

Displays properties of the unit selected with v. It also displays the IDs of any worn items. For deeper inspection of the unit and inventory items, see gui/gm-unit and gui/gm-editor.
