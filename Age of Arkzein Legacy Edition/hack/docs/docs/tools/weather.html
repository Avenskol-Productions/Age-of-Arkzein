weather¶

Tags: fort | armok | inspection | map

Command: weather

Change the weather.

This tool allows you to inspect or control the weather.

Usage¶

weather

Print a map of the local weather.

weather clear|rain|snow

Change the weather as specified.

Examples¶

weather clear

Make it stop raining/snowing.
